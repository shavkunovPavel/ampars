# README #

* ampars




#### How to use.
    cd ampars

    ./run i
        - Install go packages
    ./run b
        - Compile server
    ./run
        - Start server on 7777 port

    Run a rest client
    http://localhost:7777/requests
        - address
    Set Body content type #application/json
    Set json like #ampars/src/ampars/urls.json
    Press Send button
